
## While

```js
while(condition) {
	body;
}
```

## Do-while

```js
do {
	body;
} while (condition);
```

## For

```js
for (let i = 0; i < 5; i++) {
	body;
}
```

## For/of

```js
let programmingLanguages = ["Java", "Js", "Python"];

for (let language of programmingLanguages) {
	console.log(language);
}
```

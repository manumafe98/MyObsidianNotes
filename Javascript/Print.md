## console.log

```js
console.log("Printing");
```


## String interpolation

```js
let test = "Test";

console.log(`Printing with variables ${test}`)
```

## console.table

This is mainly for matrix visualization

```js
let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

  
console.table(matrix)
```

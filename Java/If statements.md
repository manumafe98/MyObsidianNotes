
if statements in [[Java]]

### How to use them

```java
int age = 61;

if (age >= 60) {
	System.out.println("You're a boomer");
} else if (age >= 18) {
	System.out.println("You're an adult");
} else {
	System.out.println("You're not an adult");
}
```

### Logical operators

Used to connect two or more expressions
- ``&&`` == and
- ``||`` == or
- ``!`` == not
